Attribute Test

Test setup

This test can be used to check how well certain attributes are machine translated and then read out by screen readers or ignored. The results should be self-explanatory.

Text in italics are the text subjects.

The next sentence is a <span> with the aria-label "Label, Good day." and the a text node of "Good night". Good night.

The next sentence is a <span> with the aria-description "Description: This is a description." and the a text node of "This is is text". This is Text.

The next sentence is a <span> with aria-labelledby linking to the next sentence and the a text node of Good night. Good night. Labelby: Good day.

The next sentence is a <span> with aria-describedby linking to the next sentence and the a text node of "This is text". This is text. Describedby: This is a description.

Next will be a <button> with the text "Button" and a useless but often used title-attribute of "Button description".

Lastly an image with the alt-text of "A man holding and looking intently at a small kitten. The kitten looks bored into the camera." The photo is from pixelbay.
